Reflection / Learning Experience

Learning programming using Visual Studio Code has been one of the most meaningful experiences in my academic journey. At first, I found coding difficult because of syntax errors and unfamiliar commands. However, through patience, practice, and determination, I slowly learned how to create simple programs and web pages.

This experience helped me improve my problem-solving skills, creativity, and confidence in using technology. I realized that mistakes are part of learning, and every error teaches me how to become a better programmer and student.

Overall, learning coding inspired me to continue improving my skills and motivated me to pursue a future in Information and Communication Technology.

“Programming is about learning, improving, and never giving up.”
